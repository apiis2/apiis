3. Generelle Konfiguration

Die generellen Konfigurationsparameter, wie z.B. die Größe, Farben und Schriftarten der Maske, werden in dem in Abbildung 3 gezeigten Fenster vorgenommen. Bis auf Höhe (Height) und Breite (Width) der Maske sind die anderen Parameter optional.


Abbildung 3


Hier nun eine Beschreibung der einzelnen Parameter:

Title

Der hier eingetragenen Text erscheint auf der Maske als zentrierte Überschrift in einem separaten Rahmen. Bleibt das Feld leer, erscheint auch kein Rahmen für den Titel. Statt einer Überschrift kann auch ein GIF-Bild auf der Maske plaziert werden, oder ein Text-Label (Typ A).

Model File

Die Angabe des Modelfiles ist optional. Wird kein Modelfile, oder eine nicht existierende Datei angegeben, wird beim Erstellen der Maske ein kurzer Hinweis gegeben. Mit Hilfe des links neben dem Feld angeordneten Knopfes kann eine Datei-Dialog-Box geöffnet werden.

Geometry

Die Werte für die Höhe und Breite der Maske müssen angegeben werden. Die Einheit ist Pixel.

Color

Die Überschrift und die Label der Eingabefelder werden in der Vordergrundfarbe (Foreground) dargestellt. Background ist die Farbe des Maskenhintergrunds. Werden keine Farbwerte eingetragen, sind die Farben systemabhängig.
Die Farbe für Error flash, eine Fehleranzeige, wird standartmäßig auf rot gesetzt.
Mit Hilfe der kleinen Knöpfe neben den Feldern kann ein Dialog zur Farbwahl aufgerufen werden.

Fonts

Die voreingestellten Schriftarten sind alle Helvetica. Die mit Normal bezeichnete Schriftart bezieht sich u.a. auf die Eingabefelder, während die Schriftart für die Feldbeschriftung von Label bestimmt wird.
Button-Schriftart bestimmt die Schrift für die untere Knopfleiste der Maske.
Mit Hilfe der kleinen Knöpfe kann ein Dialogfenster für die Schriftartauswahl aufgerufen werden. Da dieses nur für X11-Schriftarten möglich ist, geht das nur auf Unix-Systemen.
Es wird empfohlen, die zusätzliche Beschreibung über Fonts zu lesen.

Query Options

Logische Verbindung der einzelnen WHERE-clauses beim Suchen, welche aus den Felddaten gebildet werden.
Die Suchergebnisse können nach bestimmten Spalten sortiert ausgegeben werden. Im Feld Order by können durch Komma getrennt die Spaltennamen spezifiziert werden.

Balloons

Ballons bieten die Möglichkeit kurze Hilfetext oder Hinweise zu den Maskenobjekten in einem kleinen Fenster oder in einer Statuszeile auszugeben. Befindet sich der Mauszeiger über einem Objekt, für das ein Hilfetext angegeben ist, erscheint dieser Text und bleibt sichtbar bis der Mauszeiger sich nicht mehr über dem Objekt befindet.

Balloon none: Keine Anzeige der Balloonmeldungen, auch wenn für die Objekte Meldungen spezifiziert sind.
Balloon balloon: Die Texte erscheinen nur in einem kleinen Balloon-Fenster.
Balloon status:Die Text werden in einer Textzeile ausgegeben. Das Feld muss auf die gleich Weise wie andere Felder erstellt werden. Damit das Feld als Balloon-Status-Feld identifiziert werden kann, muss der Objektname (Sektionname) BalloonStatusField sein. Dazu muss im Konfigurationsdialog das Objekt umbenannt werden. Durch Klick auf die obere rechte graue Box (sec...) öffnet sich ein Fenster zum Umbenennen.
Balloon both: Der Balloon-Text wird in einem Balloon-Fenster und in einer Statuszeile angezeigt.
Balloon wait: Spezifiziert die Wartezeit, bevor ein Balloon erscheint. Die Wartezeit gilt für Balloon balloon und Balloon status. Wert in Millisekunden. Die voreingestellte Zeitverzögerung beträgt 350 Millisekunden.
Balloon Background Color: Spezifiziert die Hintergrundfarbe eines Balloon-Fensters. Die voreingestellte Farbe ist Gelb. Der Knopf öffnet ein Dialogfenster zur Farbwahl.
Balloon Message: Text der erscheint, wenn sich der Mauszeiger im Titelrahmen befindet.
Der kleine Knopf [...] öffnet einen einfachen Editor zum Erstellen längerer Texte.


Startup Command

Führt die spezifizierten Perl-Befehle vor dem Starten der Maske aus. Mit der system-Funktion können auch Systemkommandos ausgeführt werden.

Hier einige Beispiele:
$$form{GENERAL}{BUTTONBAR}->destroy; Entfernt die untere Knopfleiste
query($form) Dieselbe Funktion wie klick auf Suchen.
$$form{GENERAL}{TOPLEVEL}->Dialog(-text=>"Hallo")->Show; Öffnet ein Dialog-Fenster mit Text Hallo und einem OK-Knopf.
system("myShellScript"); Führt das Shell-Script 'myShellScript' aus


Siehe auch Perl-Code in Maskenparametern

Knöpfe

Create Form  Erstellt oder aktualisiert die Maske. Geht auch mit Tastenkombination Strg+u.

Close  Schließt das Konfigurationsfenster. Kann jederzeit wieder mit Edit->GENERAL geöffnet werden.

Clear  Löscht alle Felder des Konfigurationsfensters.
